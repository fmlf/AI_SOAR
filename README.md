# AI를 활용한 보안관제 시스템

> 공격 탐지 및 분석: Gemini API 프롬프트 최적화를 통한 로그 자동 분석 및 보안 위협 식별
> 대응 자동화 및 안정성 확보: AI가 생성한 대응 스크립트에 대한 관리자 검증/승인 메커니즘 적용 (오탐 방지)
> 실시간 알림 체계: Discord Webhook API를 활용한 보안 이벤트 실시간 전파 및 가시성 확보


> 사용한 툴/언어 : Discord / Jupyter-lab / Python / CentOS Stream 9

---

## 기능

Netmiko 라이브러리를 통해 장비에 SSH로 접속하여 공격행위를 시도하는 IP 차단. 
공격행위가 5회 초과일 경우 Discord에 빨간색으로 경고 알림 전송. 5회 미만일 경우 초록색 알림 전송.
관리자가 명령어를 승인 시, 명령어가 제대로 적용됐을 경우 Discord를 통해 성공 알림 전송.
관리자가 명령어 미승인 시, 명령어를 폐기처리하여 명령어가 삭제됐다는 알림 전송.
